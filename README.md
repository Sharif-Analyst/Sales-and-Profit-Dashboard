# 📊 Sales Tracker Dashboard

An interactive Excel dashboard designed to analyze sales performance, regional insights, order status, and top-selling categories.

---

## 🚀 Project Overview

This dashboard helps businesses monitor:

- Total Profit
- Regional Sales Performance
- Order Status
- Monthly Sales Trends
- Top Sellers
- Product Categories

The dashboard is fully interactive and user-friendly.

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Conditional Formatting
